## Begin

__ShadCN UI__
- Initialize ShadCN UI
    > npx shadcn@latest init

- Add the components:
    > npx shadcn@latest add
    > Select `All`

__Cursor__
- Download and Install `Cursor` (Windows, MacOS)
- `Sign In` to Cursor
- `Open` this `project` directory in `Cursor` (Click on `Editor Window`)
